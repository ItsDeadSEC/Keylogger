Simple Keylogger

This is a simple keylogger script written in Python using the pynput library. It logs keystrokes to a hidden folder on the user's computer.

## Features
- Logs keystrokes to a hidden folder in the user's AppData directory.
- Hides the console window when executed.
- Formats keystrokes for readability.

## Usage
1. Clone the repository or download the source code.
2. Ensure you have Python installed on your system.
3. Install the required dependencies using `pip install -r requirements.txt`.
4. Run the script using Python: `python keylogger.py`.
